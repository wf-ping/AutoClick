# 点击

Switch to English README `/docs/README.en.md`

用于辅助获取鼠标坐标、按间隔在指定位置自动点击的 Python 小工具集。支持命令行与图形界面，适合需要重复点击或精确定位的场景。

在 macOS 需要根据提示，在 **系统设置 → 隐私与安全性 → 辅助功能** 中对运行载体（如终端、IDE）开启权限，允许应用程序控制电脑。

项目的开发和文档翻译工作由 cursor 辅助
---

## 功能概览

| 工具 | 说明 |
|------|------|
| **mousePos.py** | 获取当前鼠标坐标，支持实时监控；带交互式 Yes/No 引导。 |
| **intervalClickGui.py** | 图形界面：可拖动圆圈标定点击位置，按间隔自动点击，支持次数与移出停止。 |

## 环境要求

- Python 3.8+
- 依赖见 `requirements.txt`

### 安装依赖

```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

**macOS 说明**：自动点击需要「辅助功能」权限。若点击无效，请在 **系统设置 → 隐私与安全性 → 辅助功能** 中为**终端**或**运行本脚本的IDE**勾选允许,否则会有两个问题：
1. 点击开始点击按钮后，鼠标不能自动转移到圆心中，因此在自动点击前触发鼠标位置检测，发现鼠标不在圆中，就是自动停止
2. 无法触发点击能力

## 一、mousePos.py — 鼠标位置

获取当前鼠标在屏幕上的坐标 (x, y)，并可进入实时监控。

### 特性

- **单次获取**：运行即输出当前坐标。
- **实时监控**：可选进入持续刷新坐标的模式（默认 0.1 秒间隔）。
- **交互式引导**：默认询问「是否进入实时监控」，支持 `[Y/n]`，回车即选默认。
- **退出时重置终端光标**：监控用 `Ctrl+C` 退出后恢复终端光标状态。

### 用法

```bash
# 打印一次位置，并询问是否进入监控
python mousePos.py

# 询问后进入监控模式
python mousePos.py -w

# 仅打印位置，不询问
python mousePos.py -y

# 直接进入监控，不询问（适合脚本调用）
python mousePos.py -w -y

# 监控时刷新间隔 0.5 秒
python mousePos.py -w -i 0.5

# 安静模式，少输出
python mousePos.py -q

# 查看完整帮助
python mousePos.py -h
```

### 参数

| 参数 | 说明 |
|------|------|
| `-w`, `--watch` | 进入实时监控模式。 |
| `-i`, `--interval SEC` | 监控时的刷新间隔（秒），默认 0.1。 |
| `-y`, `--yes` | 跳过所有询问，直接执行。 |
| `-q`, `--quiet` | 不显示使用提示，只输出结果。 |

## 二、intervalClickGui.py — 间隔点击（图形界面）

在屏幕上用圆圈标定点击位置，按设定间隔自动左键点击；可设点击次数、移出圆圈自动停止等。

### 特性

- **圆圈 + 操作框分离**
  - **圆圈**：小窗口，圆心即将要点击的坐标；可拖动到任意位置。
  - **操作框**：设置间隔、次数，开始/停止/关闭；可拖动顶部标题条移动。
- **点击逻辑**
  - 每次点击前会**暂时隐藏圆圈**再执行点击，确保点击落在下层窗口（如按钮），而不是点在圆圈上。
  - 点击间隔有**下限 50 毫秒**，避免设置过小。
- **运行中圆圈为“贴纸”**
  - 开始点击后，圆圈变浅、变为鼠标穿透，仅作位置提示；停止后恢复可拖动与正常颜色。
- **鼠标移出圆圈即停止**
  - 运行中会按一定间隔检测鼠标是否仍在圆圈内；**检测间隔自动小于点击间隔**（约为点击间隔的 1/3，且介于 20～100 ms），便于在两次点击之间移开鼠标触发停止。
- **间隔与次数**
  - **间隔（秒）**：数字输入，带加减按钮，范围 0.05～3600 秒。
  - **点击次数**：数字输入，带加减按钮；**0 表示不限**，直到手动停止或鼠标移出圆圈。
- **界面细节**
  - 间隔与次数的输入框右侧为**加减按钮**（绿色 + / 红色 -），中间一条短线分隔，符号加粗。
  - 输入框数字颜色与圆心坐标一致（橙色）。
  - 使用 Fusion 样式，各平台均显示上下两个增减按钮。
- **初始位置**
  - 操作框与圆圈首次出现时有默认位置，均可单独拖动到任意位置。
- **置顶**
  - 操作面板支持置顶/非置顶切换

### 用法

```bash
python intervalClickGui.py
```
